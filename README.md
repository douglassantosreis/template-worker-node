# Template de Woker

Template de readme para workers.


# Rodando o Projeto

1. Instalar as dependências:
    ```
        npm install
    ```
1. Rodando local
    ```
        npm run dev
    ```
# Testes

## Unitários

1. Rodar o comando: 
    ```
        npm run test
    ```

## Debug de testes

1. Atachar o debugger
1. Rodar o comando: 
    ```
        npm run test:debug
    ```


## Mutantes

1. Rodar o comand:
    ```
        npm run test:mutation
    ```
